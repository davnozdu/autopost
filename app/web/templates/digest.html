Настройки

«Новости» — пул RSS + Brave. «Новинки кино» — свежие релизы с торрентов (Torznab: Jackett/Prowlarr), рейтинги/постеры OMDb, magnet-ссылки уходят первым комментарием. Кино — только Telegram.

Ранжирование по актуальности (Brave Search)

Brave сортирует пул по актуальности темы — без расхода токенов LLM (1 запрос). В режиме «Новости» — по свежести/важности. В режиме «Новинки кино» — поднимает релизы, которые сейчас на слуху (премьеры, о которых пишут), и опускает старые ре-апы; тема, напр. «новые фильмы сериалы премьеры 2026».

Новинки кино (только режим «Новинки кино»)

Источники берутся автоматически из Prowlarr (все индексаторы разом) — вести список URL не нужно. Просто отметь, что публиковать; робот возьмёт свежие релизы этих категорий, свернёт дубли, добудет magnet и опубликует топ 2–5 (поле «сколько в LLM») по свежести.

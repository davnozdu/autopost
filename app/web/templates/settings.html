Бот мониторинга (Telegram)

Бот шлёт вам в личку ошибки публикации/входа сразу и раз в день — сводку по площадкам, а также принимает команды управления. Создайте отдельного бота у @BotFather (не тот, что публикует — иначе конфликт getUpdates), напишите ему любое сообщение, затем укажите токен и ваш chat_id (узнать у @userinfobot).

Команды в боте: /status — состояние и очереди · /stats — сводка · /post — публикация мастером · /ig /tg /x /all текст — опубликовать вручную вне очереди (можно с временем: /tg 18:30 текст, /all +30m текст; фото — картинкой с подписью). /help — справка.

Кнопки работают по уже сохранённым настройкам.

Резервный LLM (если основной недоступен)

Если основной провайдер (DeepSeek) перестаёт отвечать, обработка временно переключается на резервный канал — по умолчанию OpenAI/ChatGPT по API. Это именно резерв с авто-восстановлением: при сбое запросы на {{ (fallback.seconds_left // 60) if fallback else 5 }}≈5 минут идут в резерв, затем основной пробуется снова и, как только отвечает, всё возвращается на него.
